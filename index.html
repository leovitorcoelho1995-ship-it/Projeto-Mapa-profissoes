Leitura principal

Python, Machine Learning, AWS e SQL aparecem com alta demanda. O Opportunity Score combina volume de vagas, crescimento em buscas e interesse atual, criando uma régua simples para priorizar estudo e posicionamento profissional.

Cuidados metodológicos

Salários são estimados por referência externa, porque só 2 vagas informaram remuneração. Google Trends mede interesse relativo, não contratação direta. O score é um indicador criado para comparação, não uma verdade de mercado.

CBOS_TI = [212205, 212210, 212215, 212305,
           212310, 212315, 212405, 317110, 142305]

def carregar_mes(path: Path) -> pd.DataFrame:
    df = pd.read_csv(
        path,
        sep=";",
        encoding="utf-8",
        usecols=COLUNAS,
        dtype={"cbo2002ocupacao": str},
        on_bad_lines="skip"
    )
    return df[df["cbo2002ocupacao"].astype(float).isin(CBOS_TI)]

Decisão de design

O filtro por CBO acontece antes das agregações. Isso reduz memória, evita carregar milhões de linhas irrelevantes e transforma microdados pesados em CSVs gold pequenos o suficiente para uso direto no Power BI.
