02_processa.py 
CBOS_TI = [212205, 212210, 212215, 212305,
           212310, 212315, 212405, 317110, 142305]

def carregar_mes(path: Path) -> pd.DataFrame:
    df = pd.read_csv(
        path,
        sep=";",
        encoding="utf-8",
        usecols=COLUNAS,
        dtype={"cbo2002ocupacao": str},
        on_bad_lines="skip"
    )
    # filtro acontece AQUI — antes de qualquer operação pesada
    return df[df["cbo2002ocupacao"].astype(float).isin(CBOS_TI)]

def gerar_gold(df: pd.DataFrame) -> None:
    # série temporal de saldo líquido
    (df.groupby("ano_mes")["saldomovimentacao"]
       .sum().reset_index()
       .to_csv(GOLD_DIR / "serie_temporal.csv", index=False))

    # salário mediano e contagem por ocupação
    (df[df["salario"] > 0]
       .groupby("cbo2002ocupacao")["salario"]
       .agg(mediana="median", media="mean", n="count")
       .reset_index()
       .to_csv(GOLD_DIR / "salario_por_cbo.csv", index=False))
Por que filtrar durante o parse e não depois? O pandas aloca memória para cada linha lida. Filtrar após carregar tudo significa manter ~50M de linhas na RAM só para descartar 98% delas. Filtrar no read_csv mantém o pico de memória proporcional ao resultado, não ao arquivo.

Wayto Tech —
ferramentas para
consultores TOTVS

Consultores TOTVS perdem horas toda semana em tarefas que deveriam ser resolvidas em minutos — buscar parâmetros em documentação dispersa, montar queries do zero, migrar dados entre sistemas sem validação.

O Wayto Tech é uma plataforma SaaS construída especificamente para esse nicho: glossário de tabelas Protheus, biblioteca de queries SQL, base de erros conhecidos, migração de dados com auditoria campo a campo, e gestão de clientes e projetos.

Em construção ativa. O projeto CAGED Analytics foi desenvolvido em paralelo como exercício das mesmas habilidades — Python, SQL, BI — que sustentam a engenharia por baixo do Wayto.

Limpeza e validação de dados

Módulo de data cleansing com validação de CPF, CNPJ, datas, duplicatas e campos obrigatórios. Indispensável antes de qualquer migração ou carga em produção.

CPF / CNPJ dedup relatório de qualidade
Por que TOTVS? O ERP domina ~50% do mercado brasileiro de médias e grandes empresas. A documentação oficial é fragmentada, as queries precisam ser remontadas a cada projeto, e não existe uma ferramenta consolidada para esse workflow. O Wayto resolve exatamente esse gap — não é um produto genérico, é vertical por design.
