📋 Sobre o projeto

Dashboard que cruza dados de vagas reais (Adzuna API) com tendências de busca de 5 anos (Google Trends) para identificar as skills e profissões mais promissoras do mercado de tecnologia no Brasil.

🔧 Tecnologias utilizadas

Python Adzuna API Google Trends Pandas Streamlit Plotly NLP

💡 Principais descobertas

IA cresceu +2075% em buscas nos últimos 5 anos
Python é a skill mais demandada (93 vagas)
DevOps tem o melhor equilíbrio demanda/salário
83% das vagas não especificam modalidade de trabalho
